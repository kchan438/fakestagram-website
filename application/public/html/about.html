About


This website is created by Kevin Chan and is intended for academic purposes only. This web application is created using HTML, CSS, nodeJS, ExpressJS, and MySQL. In this website, we've created a website which stores images posted by users. Each user must create an account in order to post an image. Users can also comment on other user's posts as well.
